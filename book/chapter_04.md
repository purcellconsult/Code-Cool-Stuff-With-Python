Since the game counts down a total of 5 seconds, the loop will cycle 5 times and during each iteration the <code>sleep</code> function is called; the 1 pauses the program for one second. Since the condition is <code>while secs &gt; 0</code>, make sure to decrement the counter by 1 each cycle so that it will indeed reach 0. The next step is to randomly set the speeds of the turtles:</p>
<pre><code>def set_turtle_speeds(self):
  
 speeds = [gait for gait in range(1, 25)]
 self.red_speed = choice(speeds)
 self.blue_speed = choice(speeds)
 self.green_speed = choice(speeds)
 self.black_speed = choice(speeds)
</code></pre>
<p>To make this game unpredictable the turtles will be randomly assigned a speed from 1 to 25. A turtle that gets assigned a low speed like 1 will truly mimic a turtle in real life!</p>
<p>Let‚Äôs next dig into the logic required to make the turtle race and then determine the actual winner. Below is the beginning snippet for this:</p>
<pre><code>def turtle_race(self):
  
    race_on = True
    while race_on:
        if self.red.pos()[0] &gt;= self.finish.pos()[0]:
            if self.blue.pos()[0] &gt;= self.finish.pos()[0] and self.blue.pos()[0] &gt; self.red.pos()[0]:
                self.finish.penup()
                self.finish.goto(-self.width, self.height / 2.9)
                self.finish.pendown()
                self.finish.write('Blue Wins!', font=('Verdana', 13))
                race_on = False
                return self.blue
            elif self.green.pos()[0] &gt;= self.finish.pos()[0] and self.green.pos()[0] &gt; self.red.pos()[0] :
                self.finish.penup()
                self.finish.goto(-self.width, self.height / 2.9)
                self.finish.pendown()
                self.finish.write('Green Wins!', font=('Verdana', 13))
                race_on = False
                return self.green
            elif self.black.pos()[0] &gt;= self.finish.pos()[0] and self.black.pos()[0] &gt; self.red.pos()[0]:
                self.finish.penup()
                self.finish.goto(-self.width, self.height / 2.9)
                self.finish.pendown()
                self.finish.write('Black Wins!', font=('Verdana', 13))
                race_on = False
                return self.black
            self.red.forward(25)
            self.finish.penup()
            self.finish.goto(-self.width, self.height / 2.9)
            self.finish.pendown()
            self.finish.write('Red Wins!', font=('Verdana', 13))
            race_on = False
            return self.red
</code></pre>
<p>In this code snippet the method is defined and a while loop is created with the flag <code>race_on</code> which is initially set to <code>True</code>. The while loop will keep cycling indefinitely until race_on is set to <code>False</code> which happens when the winning turtle crossed the finish line. The next question is how do we determine that?</p>
<p>Every turtle object that we‚Äôve drawn up until this point has an x and y position, just like on the Cartesian Coordinate system. We can access the current x or y value of a turtle object by invoking the <code>pos</code> method. So, if we want the current x value of the red turtle we can do this:</p>
<pre><code>self.red.pos()[0] 
</code></pre>
<p>If we want the y position of the red turtle we can do that:</p>
<pre><code>self.red.pos()[1]
</code></pre>
<p>However, we need something to compare it to. Since we want to know which turtle won we should compare the current x value of each turtle to the x value of the finish line. That‚Äôs where this snippet of code comes in handy:</p>
<pre><code>if self.red.pos()[0] &gt;= self.finish.pos()[0]:
</code></pre>
<p>At the end of each iteration we need to progress the turtles forward which we can accomplish by using the following code snippet:</p>
<pre><code>self.red.forward(self.red_speed)
self.blue.forward(self.blue_speed)
self.green.forward(self.green_speed)
self.black.forward(self.black_speed)
</code></pre>
<p>You would think that this is all we needed to determine the winning turtle but there‚Äôs something we‚Äôre missing. For example, let‚Äôs assume that the red turtle speed was 20, and let‚Äôs also assume that the red turtle has crossed the finish line. While the red turtle may indeed cross the finish line, there‚Äôs a couple of edge cases we need to check. In order to fully crown Mr. or Mrs. Red Turtle the winner, we need to check that the other three turtles don‚Äôt have a speed that‚Äôs greater than 20.</p>
<p>If they do, then what could happen is that our program has a bug in it because the red turtle could be declared the winner when it doesn‚Äôt actually have the highest speed.</p>
<p>Let‚Äôs still assume that the red turtle again has a speed of 20, and let‚Äôs assume that the blue turtle has a speed of 22. The speeds are really close, and if we don‚Äôt add some type of conditional check then what could happen is that the red turtle could be declared the winner simply because it‚Äôs placement in code comes before that of the blue turtle.</p>
<p>That‚Äôs where those nested conditional checks come into play. For example, let‚Äôs analyze this code snippet:</p>
<pre><code>if self.red.pos()[0] &gt;= self.finish.pos()[0]:
    if self.blue.pos()[0] &gt;= self.finish.pos()[0] and self.blue.pos()[0] &gt; self.red.pos()[0]:
</code></pre>
<p><em>What this says in English is if the x value of the red turtle is greater than the x value of the finish line; and if the position of the blue turtle is greater than the x value of the finish line and the x position of the blue turtle is greater than the x position of the red turtle then‚Ä¶</em></p>
<p>The code that comes afterwards positions the pen above the finish line, sets <code>race_on</code> to <code>False</code> which makes the while loop stops, and then returns the blue turtle. We need to return the winning turtle as that decides the victory dance which logic will be described next. However, we need to do these conditional checks for the green and black turtle since the red turtle appears first in the code. Only after those checks are done is it safe for us to assume that the red turtle is indeed the winning turtle.</p>
<p>At this point I also decided to include <code>forward(25)</code> on all of the winning turtles so that it will look like they sprint pass the finish line like in a n actual track race.</p>
<p>Similar logic applies to the blue, green, and black turtles with the exception that they need to check the turtles that come before them in the code. That‚Äôs the logic to start the turtles off racing and to ensure that the turtle with the fastest speed is crowned the winner.</p>
<p>The next snippet of code shows the logic on how to make the turtle do it‚Äôs silly spin dance and then make it grow in size afterwards:</p>
<pre><code>def victor_dance(self):
   
    if self.turtle_race() == self.red:
        for x in range(50):
            self.red.right(90)
        self.red.shapesize(10, 10)
        return
    elif self.turtle_race() == self.blue:
        for x in range(50):
            self.blue.right(29)
        self.blue.shapesize(10, 10)
        return
    elif self.turtle_race() == self.green:
        for x in range(50):
            self.green.right(30)
        self.green.shapesize(10, 10)
        return
    elif self.turtle_race() == self.black:
        for x in range(15):
            self.black.right(90)
        self.black.shapesize(10, 10)
        return
</code></pre>
<p>As you can see it;s simply a for loop that cycles a variable number of times and then rotate the turtle to the right a variable number of degrees. At the end of the iteration the <code>shapesize</code> method is invoked which allows the change in dimensions of the turtle. The first parameter coincides with the stretch width while the second parameter coincides with the stretch length.</p>
<p>The last portion of the code is to just execute all of the methods. The order does matter because we don‚Äôt want to call a method that‚Äôs trying to access an object that doesn‚Äôt exist. Below is the code for this:</p>
<pre><code>if __name__ == '__main__':
    message = turtle.Turtle()
    message.hideturtle()
    race = TurtleRace()
    race.start_turtles()
    race.finish_line()

    race.set_turtle_speeds()
    race.turtle_race()
    race.victor_dance()
    turtle.done()
</code></pre>
